# @viberevert/core

> Session lifecycle, config, paths, redaction.

Core primitives shared across the VibeRevert CLI and its libraries: config loader, repo-root resolver, `.viberevert/` directory helpers, and secret-redaction utilities.

Part of [VibeRevert](https://github.com/madeinplutofabio/vibe-revert) — the safety belt for vibe coding.

**Status:** `v0.7.0-beta` work in progress. Public API may change before v1.0.
