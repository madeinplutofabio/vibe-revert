# @viberevert/session-format

> Schemas and types for VibeRevert session artifacts.

Zod schemas and JSON Schema exports for `SessionReport`, `CheckResult`, `Evidence`, `ChangedFile`, and `Manifest`. The contract every other package reads and writes.

Part of [VibeRevert](https://github.com/madeinplutofabio/vibe-revert) — the safety belt for vibe coding.

**Status:** `v0.7.0-beta` work in progress. Public API may change before v1.0.
